NMIX 5110 Final Project

Making my Portfolio using Vue.js

For my final project, I chose to create a Projects page and a Photography page on my portfolio website using Vue.js.

Projects Page

On the projects page, the projects are loaded from a local JSON file hosted on the server asynchronously by a Vue instance. The JSON is parsed for all of the project details to create card views and modal views when the cards are selected. The array of projects can be filtered by project type using methods of the Vue instance.

Photography Page

On the photography page, the photos are loaded using the Flickr API which collects photos from my personal Flickr account. The first query gets the info about where the file is hosted so that it can be downloaded, and a second query uses that info to retrieve the metadata of each photo to get its title and date. The JSON returned by the Flickr API is parsed to create the card views and setup a link to view the photo at full-resolution on Flickr when clicked.
